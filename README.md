## ⚙️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/TG-office-utils.git
   ```
2. Navigate to the project directory:
   ```bash
   cd TG-office-utils
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
